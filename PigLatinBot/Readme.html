Creating an Agent Service

Important build info

To build this project you will need to add this url as a Nuget package source before you build

https://fuselabs.pkgs.visualstudio.com/DefaultCollection/_packaging/packages/nuget/v3/index.json

Getting a subscription key

  1. Go to Intercom Developer portal and get a subscription to the Intercom for Agents product.
  2. Update app.config Subscrtipion key with the value from the portal

Testing

Using a tool like PostMan submit:
POST http://localhost:yourport/api/Messages
With Basic Aauth headers
User: Appid
Password: AppSecret
and with a JSON payload like
{
    "Text":"Hello world",
}

Register your agent

  1. Register your agent with the Intercom Developer portal and get an AppId and AppSecret for your agent
  2. Update app.config with your AppId and AppSecret key with the values from the portal
  3. Test your agent with the Test Channel using your agentId
  4. Configure your agent for other channels via the Intercom Developer portal
